I have completed all the 4 requirements. The site is hosted on AWS. 

The program is made using AngularJS and bootstrap.

Steps for running on local machine.
1) Install Node.js using installation details given on node website.
2) Install express using following instrucctions
   - use command line to be the current directory of homework folder
   - run "npm install express --save"
   - express will be installed 
3) type "node app.js" from the same root folder.
4) site can be accessed using "localhost:3002"
